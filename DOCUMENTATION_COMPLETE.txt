6. FONCTIONNALITÉS DÉTAILLÉES
================================================================================

DÉTECTION MD5:

Algorithme:
1. Pour chaque fichier: calculer MD5
2. Grouper par MD5 identique
3. Fichiers même MD5 = DOUBLON EXACT
4. Calculer taille récupérable
5. Afficher résultats

Exemple:
   document.pdf (MD5: abc123...) 
   doc_copie.pdf (MD5: abc123...) ← MÊME HASH = DOUBLON

MULTILINGUE (FR/EN):

Implémentation: next-intl
   • 130+ clés de traduction
   • Changement dynamique sans recharge
   • Détection automatique du navigateur

Utilisation code:
   const t = useTranslations()
   <h1>{t('commun.titre')}</h1>

MODE SOMBRE AUTOMATIQUE:

• Détecte prefers-color-scheme du navigateur
• CSS media queries
• Bouton toggle manuel optionnel
• Stockage dans localStorage

Couleurs:
   Clair: Blanc (#fff), Noir texte (#000)
   Sombre: Noir (#0a0a0a), Blanc texte (#f0f0f0)

ANIMATIONS:

Définies dans globals.css:
   • pulse: Pulsation infinie (0.5s)
   • slide-in: Entrée latérale (0.5s)
   • fade-in: Fondu (0.3s)

Utilisation:
   • Apparition cartes KPI: slide-in
   • Indicateur chargement: pulse
   • Résultats: fade-in

GRAPHIQUES:

Recharts visualisations:
   1. Pie Chart: Distribution doublons/unique
   2. Bar Chart: Top 5 fichiers
   3. Line Chart: Tendance temporelle

   • Node.js 18+
   • SSH/Terminal accès

OPTION 3: DOCKER

Dockerfile:
   FROM node:18-alpine
   WORKDIR /app
   COPY . .
   RUN pnpm install && pnpm build
   EXPOSE 3000
   CMD ["pnpm", "start"]

Build et run:
   docker build -t app . && docker run -p 3000:3000 app

8. TROUBLESHOOTING
================================================================================

Port 3000 utilisé?
   pnpm dev -- -p 3001

Module non trouvé?
   pnpm install

Langue ne change pas?
   • F12 → Console (voir erreurs)
   • Vider cache: Ctrl+Shift+Del
   • Recharger: Ctrl+F5

Mode sombre ne fonctionne pas?
   • Vérifier paramètres système
   • Mettre à jour navigateur
   • Cliquer bouton 🌙 manuel

Scan ne démarre pas?
   • F12 → Console (erreurs?)
   • JavaScript activé?
   • Recharger page

Build error "Cannot find module"?
   • Dépendances ok? pnpm install
   • Relancer build

Performance lente?
   • F12 → Lighthouse
   • Utiliser Vercel (optimisation auto)
   • Réduire fichiers test
