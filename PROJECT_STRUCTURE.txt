🔗 VARIABLES PRINCIPALES (En français)

Services de détection:
  • fichiersTotaux: nombre de fichiers scannés
  • nombreDoublons: nombre de fichiers dupliqués
  • tailleRecuperee: espace économisable en octets
  • tauxDoublons: pourcentage de doublons
  • tempsEcoule: durée du scan en ms

Interfaces:
  • FichierMetadonnees: métadonnées d'un fichier
  • GroupeDoublons: groupe de fichiers identiques
  • ResultatScan: résultat complet du scan

🎨 DESIGN SYSTEM

Couleurs (Tailwind CSS v4):
  • Arrière-plan: blanc (#fff) / noir (#0a0a0a)
  • Texte: noir (#1a1a1a) / blanc (#f0f0f0)
  • Accent: gris foncé (#2d2d2d) / gris clair (#e8e8e8)
  • États: vert (succès), ambre (alerte), rouge (erreur)

Animations:
  • pulse: pulsation infinie
  • slide-in: apparition latérale
  • fade-in: fondu
  • Transitions: 300ms smooth

Composants:
  • .carte: boîte avec ombre
  • .btn-principal: bouton primaire
  • .btn-secondaire: bouton secondaire
  • .badge: petit label
  • .titre-principal: titre avec gradient
