⚙️ COMMANDES RAPIDES
────────────────────────────────────────────────────────────────────────────

Installation:
  $ pnpm install

Développement (avec hot reload):
  $ pnpm dev
  # Ouvrir http://localhost:3000/fr

Linter / Format:
  $ pnpm lint (si configuré)

Build production:
  $ pnpm build

Démarrer production:
  $ pnpm start

Deploy sur Vercel:
  $ npm i -g vercel
  $ vercel deploy --prod

❓ AIDE RAPIDE
────────────────────────────────────────────────────────────────────────────

Q: Où sont les fichiers de l'application?
A: /app (pages), /components (composants), /lib (services)

Q: Comment ajouter une nouvelle page?
A: Créer app/[locale]/nomPage/page.tsx et ajouter au menu

Q: Comment changer la détection?
A: Modifier lib/servicesDetection.ts - fonction detectionDoublonsExacts()

Q: Comment ajouter une langue?
A: Ajouter messages/nouvelleLangue.json et mettre à jour routing.ts

Q: Comment voir les graphiques?
A: Aller à la page "Rapports" (section rapports)

Q: L'app est lente?
A: Vérifier F12 → Lighthouse. Vercel optimise automatiquement.

Q: Je veux plus de détails?
A: Lire GUIDE_COMPLET.md (section correspondante) ou DOCUMENTATION.md
